\# 🎬 Movie Ticket Booking System



A full-stack Movie Ticket Booking System built using FastAPI (backend) and React (frontend).



\---



\## 🚀 Features



\- User Login \& Register

\- Browse Movies

\- Movie Details

\- Select Theater \& Show

\- Seat Booking

\- Stripe Payment Integration

\- Download Booking Invoice

\- Notifications

\- Booking History



\---



\## 🛠️ Tech Stack



Backend:

\- FastAPI

\- MySQL



Frontend:

\- React

\- Tailwind CSS



\---



\## ⚙️ Run Project



\### Backend

cd backend/fastapi\_app

venv\\Scripts\\activate

uvicorn main:app --reload --port 8001



\### Frontend

cd frontend

npm install

npm run dev
